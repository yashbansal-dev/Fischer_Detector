# Voting Classifier Confusion Matrix Analysis

**Graph File:** `Voting_Classifier_cm.png`

## Overview
This matrix shows the result of a "Majority Vote" across Logistic Regression, Random Forest, and SVM.

## Performance Context
*   **Accuracy:** 91.77%
*   **Precision:** 91.77%
*   **Recall:** 91.77%
*   **F1-Score:** 91.49%

## Key Observations
1.  **Consensus Accuracy:** By voting, the model smooths out individual errors from linear models.
2.  **Stable Predictions:** The confusion matrix shows a very balanced performance between benign and malicious classes.

## Conclusion
A robust ensemble method that performs very well, though the single Random Forest model still outperforms it slightly.
